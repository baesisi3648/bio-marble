## 🎮 게임 규칙

### 기본
- **보드**: 10×6 그리드, 28칸 시계방향
- **시작 코인**: 30
- **플레이어**: 2~4 모둠
- **이동**: 주사위 2개 합산 (2~12칸)
- **더블 보너스**: 같은 숫자 두 개 나오면 **⚡ 더블!** 이펙트 + **한 번 더 굴림**
- **승리 조건**: 코인이 0이 되면 파산, 마지막 1팀이 우승

### 플레이어 토큰
- 4모둠: 🦊 (빨강) / 🐯 (파랑) / 🐼 (초록) / 🐰 (주황)
- 현재 턴 플레이어 위에는 **👑 금색 펄스 스포트라이트** 표시

### 칸 효과
| 칸 | 효과 |
|----|------|
| 🏁 START 통과 | +5 코인 |
| 🏁 START 도착 | +7 코인 |
| 🏕️ 생물보호구역 | **비었으면 5코인 납부, 찼으면 누적 전액 획득** (pay-forward 누적 시스템) |
| ⛓️ 감옥 | 첫 도착: 3코인 납부 or 대기 · 다음 턴부터 주사위 더블로 탈출 가능 |
| ✈️ 세계동물여행 | 원하는 동물 칸으로 이동 (경로상 START 포함 시 +5코인) |
| 🔑 생물구조열쇠 | 14종 랜덤 카드 |

### 동물 칸 퀴즈 플로우
1. 동물 칸 도착 → **[문제 시작] 버튼** 표시
2. 버튼 클릭 → **3-2-1 카운트다운** 풀스크린
3. **풀스크린 퀴즈** 등장 + 10초 타이머 (문제 동안 ambience + 마지막 3초 긴장감)
4. 타이머 종료 → **[정답 공개] 버튼** (자동공개 X)
5. 진행자가 정답 확인 후 **[⭕성공] [❌실패]** 자가판정

### 문제 출제 방식
- 모든 문제가 **셔플된 큐**에서 하나씩 추첨
- 한 사이클(=50문제) 안에서 중복 출제 없음
- 게임 시작 시마다 큐 리셋 (새로 셔플)
- 풀 소진 시 자동 재셔플

### 동물원 (놀이공원 테마)
모든 레벨 업그레이드 시 **퀴즈 필수**

**건설비 = 지역세금 + 레벨비용 · 실패 시 지역세금만 2배**

| 레벨 | 모양 | 레벨비용 | 초원(1) 성공/실패 | 바다(3) 성공/실패 |
|------|------|----------|-------------------|-------------------|
| 🎡 Lv.1 신규 건설 | 관람차 | 3 | 4 / 5 | 6 / 9 |
| 🎢 Lv.2 업그레이드 | 롤러코스터 | 4 | 5 / 6 | 7 / 10 |
| 🏰 Lv.3 무적 업그레이드 | 성 | 5 | 6 / 7 | 8 / 11 |

- Lv.2에는 🛡️(파란 glow) 배지 — 인수 불가 표시
- Lv.3에는 🛡️(금색 glow) 배지 — 인수 불가 + 생물구조열쇠 가로채기/폐쇄/교체 보호

### 관람비 (상대 동물원 방문)
- 관람비 = `(지역세금 + 2) × Lv배수` (Lv.1 ×1, Lv.2 ×2, Lv.3 ×3)
- 실패 시: `관람비 + 3코인 flat` 추가
- **퀴즈 성공 + Lv.1 동물원**일 때만 **2배 지불로 인수** 가능
- Lv.2 이상은 인수 불가 (관람비만 지불)
- 인수해도 레벨은 Lv.1 유지 (업그레이드 X)

### 🔑 생물구조열쇠 (14종 랜덤 카드)
| 카드 | 효과 |
|------|------|
| 상금 +2 / +4 / +6 / +8 코인 | 코인 획득 |
| 벌금 -2 / -4 코인 | 코인 차감 |
| 원하는 곳으로 이동 | 텔레포트 (경로상 START 통과 시 +5코인) |
| 상대 동물원 가로채기 | Lv.3 제외 상대 동물원 1개 강탈 |
| 동물원 교체 | 상대 동물원과 내 동물원 맞교환 |
| 상대 동물원 폐쇄 | Lv.3 제외 상대 동물원 1개 파괴 |
| 무상 동물원 설립 | 빈 동물 칸에 무료 Lv.1 건설 |
| 주사위 두 번 | 다음 턴 주사위 추가 1회 |
| **다른 팀 감옥 보내기** | 대상 선택 → 감옥행 + 2턴 정지 |
| **다른 팀과 코인 교환** | 각 팀 코인 표시된 피커로 선택 (자기 팀 포함, 다른 팀 선택 시 코인 총액 맞교환) |

## 🚀 배포 — Cloudflare Pages (GitHub 연동)

### 1. Git 저장소 초기화
```bash
cd C:/Projects/bio_marble
git init
git add .
git commit -m "Initial commit"
```

### 2. GitHub 레포 생성
- github.com에서 `bio-marble` 레포 생성 (Public 또는 Private)
```bash
git remote add origin https://github.com/<USERNAME>/bio-marble.git
git branch -M main
git push -u origin main
```

### 3. Cloudflare Pages 연결
1. dash.cloudflare.com → **Workers & Pages** → **Create** → **Pages** → **Connect to Git**
2. GitHub 계정 인증 → `bio-marble` 레포 선택
3. **Build settings**:
   - Framework preset: **None**
   - Build command: (비워둠)
   - Build output directory: `/`
4. **Save and Deploy**

### 4. 접속
- 게임: `https://bio-marble.pages.dev/`
- 관리자: `https://bio-marble.pages.dev/admin.html`

### 수정 반영
- 로컬에서 수정 → `git add . && git commit -m "..." && git push`
- Cloudflare가 자동 재배포 (1~2분)

> ℹ️ 스크립트에 `data-cfasync="false"`를 달아 Cloudflare Rocket Loader의 async 재작성을 방지합니다.
